# 大大物流比价助手 使用说明

## 功能特点
1. 实时比价：支持顺丰、圆通、中通、京东、韵达等主流快递
2. 智能计算：考虑重量、目的地、紧急程度三个维度
3. 数据透明：显示各家优势和劣势，帮你做出最佳选择

## 如何使用
1. 输入货物重量（支持滑块和数字输入）
2. 选择目的地类型
3. 选择紧急程度
4. 点击"立即比价"

## 数据说明
- 价格数据基于2026年2月市场调研
- 实际费用可能因促销、区域政策有所不同
- 偏远地区包含：新疆、西藏、青海等
