## 📌 项目简介
**一款基于真实物流经验开发的快递比价工具**  
输入重量、目的地、紧急程度，即可快速对比物流快递公司的预估价格及服务特点。

## 🧠 设计背景
- 作者拥有 **12年一线物流从业经验**
- 深知物流从业者在报价环节的痛点：**查价慢、比价烦、信息不透明**
- 本工具旨在用最简单的方式，**3秒内给出多家报价对比**

## ✨ 核心功能
- ✅ 实时比价：（物流公司信息尚未录入）
- ✅ 三维度计算：重量 / 目的地 / 紧急程度
- ✅ 直观卡片展示：价格、优势、劣势、时效一目了然
- ✅ 移动端适配：手机也能流畅使用

## 🛠️ 技术栈
- **前端**：HTML5、CSS3、JavaScript（原生，零依赖）
- **设计**：AI辅助生成Logo及视觉元素
- **部署**：GitHub Pages（永久免费）

## 📊 数据来源
- 基础价格参考**2026年2月市场公开报价**
- 后续将根据**前物流同事的真实反馈**持续更新
